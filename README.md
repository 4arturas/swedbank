# Swedbank Bank

## Backend

```bash
cd backend
set JAVA_HOME=...
mvn spring-boot:run
```

Starts on `http://localhost:8080`. 
- H2 console: `/h2-console`
- Swagger UI: `/swagger-ui.html`

## Frontend

```bash
cd frontend
ng serve
```

Starts on `http://localhost:4200`. Open in browser to use the app.

Users: Alice (ID: 1), Bob (ID: 2).
